## Indexded db : database in html5
Form to Insert information of corporation members into **persistance database client computers**.

#### Libraries and Dependencies
- [IndexedDBShim](https://github.com/axemclion/IndexedDBShim)
- [Materialize](http://materializecss.com/)
- [Pickadate](http://amsul.ca/pickadate.js/date/)
- [Bower](http://bower.io/)
